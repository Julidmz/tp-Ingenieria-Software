
# En git corren tambien los comandos de linux

# Si modifico el archivo tengo que volver a comitearlo para confirmar la modificacion

 1 - Eliminamos la version de git q teniamos e instalamos la version actualizada q es la 2.44

 2 - Para corroborar, vas al cmd y pones: git -- version

 3 - Abrimos git y creamos nuestro nombre:  git config --global user.name "Julian"

 4 - Creamos nuestro mail: git config --global user.email julidmz46@gmail.com

 5 - Para confirmar que fue creado ponemos: git config --list
     y al final de la lista tiene q aparecer nuestros datos volcados

 6 - Creamos una carpeta dentro de documentos q se va a llamar git2, y adentro de esta carpeta creamos otra carpeta que se va a llamar 
     proyecto1 y adentro de esta carpeta creamos un bloc de nota: readme.txt

 7 - Presionamos click derecho adentro de la carpeta proyecto1 y vamos a la opcion: open git bash here

 8 - Una vez adentro iniciamos el proyecto:  git init

 9 - Para ver el estado actual del repositorio utilizamos: git status

 10 - Para hacer un cambio: git add readme.txt

 11 - Para volver a corroborar el estado: git status

 12 - Para confirmar el cambio: git commit -m "inicio de proyecto"

 # Si no le pongo -m abre el visual studio porque le indicamos que es el editor por defecto.

 13 - Para clonar un proyecto: git clone 

 14 - Para crear rama + el nombre de la rama: git branch "testing"
